# SiPH Remotion Video Demo

This is a small Remotion proof-of-concept for exporting the SiPH simulator as a coded MP4 video.

## Run Studio

```bash
npm install
npm run dev
```

## Render MP4

```bash
npm run render
```

The video renders to:

```text
out/siph-coupling-demo.mp4
```

## Render Preview Still

```bash
npm run render:png
```
